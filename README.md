# Desktop Beleza

**Desktop Beleza** é um sistema de gestão desenvolvido para o salão de beleza do Senac DF - Unidade Taguatinga. Ele resolve o problema de agendamento de serviço para a comunidade. O objetivo é otimizar esse agendamento.

## Funcionalidades

- Cadastro de clientes/modelos
- Agendamento de serviços por horário
- Controle de turmas e instrutores

## Tecnologias

- **Java** - Linguagem principal
- **JavaFX** - Interface gráfica
- **MySQL** - Banco de dados
- **JDBC** - Conexão com banco de dados
- **AtlantaFX** - Estilização dos elementos da interface
- **jBCrypt** - Criptografia
- **IntelliJ IDEA** - IDE

## Instruções de Uso

1. **Pré-requisitos**
    - Java 25+
    - MySQL Server

2. **Clonar Repositório**
    ```bash
    git clone https://github.com/instrutorAdriel/Desktop_Beleza
    ```
   
3. **Abrir o projeto em uma IDE** (Recomendado utilizar IntelliJ IDEA)

4. **Executar o Launcher.java**
